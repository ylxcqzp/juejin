## 2. 微服务拆分与职责

**初期采用5个核心服务，降低复杂度：**

| 服务名 | 职责描述 | 主要存储依赖 |
|--------|----------|--------------|
| gateway-service | 统一入口，路由转发，鉴权校验，跨域处理，限流 | Nacos |
| user-service | 用户认证、用户信息、关注粉丝、成长体系 | MySQL, Redis |
| content-service | 文章管理、标签体系、专栏、搜索、审核流程 | MySQL, ES, Redis |
| social-service | 点赞、收藏、评论、沸点、Feed流、消息通知 | MySQL, Redis |
| operation-service | 签到任务、热榜计算、文件资源、定时任务调度 | MySQL, Redis, OSS |

**后期可拆分：**
- content-service -> content-service + search-service
- social-service -> social-service + message-service + feed-service
- operation-service -> operation-service + resource-service + job-service

## 3. 前端架构设计

### 3.1 C端UI方案选择

**推荐方案：Headless UI + Tailwind CSS**

| 对比项 | Element Plus | Headless UI + Tailwind |
|--------|--------------|------------------------|
| 样式定制 | 需覆盖CSS变量，难度大 | 完全自定义，灵活度高 |
| 包体积 | 较大（全量引入） | 按需引入，体积小 |
| 设计一致性 | 组件风格固定 | 完全可控，品牌感强 |
| 开发效率 | 高（组件丰富） | 中（需自行封装） |
| 适用场景 | 管理后台 | C端用户界面 |

**结论：** C端页面追求独特设计和用户体验，建议使用 **Headless UI + Tailwind CSS**，管理后台使用 Element Plus。

### 3.6 Axios 封装

```typescript
// api/request.ts
import axios from 'axios'
import { useUserStore } from '@/stores/user'

const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || '/api/v1',
  timeout: 10000,
  headers: {
    'Content-Type': 'application/json'
  }
})

// 请求拦截器
request.interceptors.request.use(
  (config) => {
    const userStore = useUserStore()
    if (userStore.token) {
      config.headers.Authorization = `Bearer ${userStore.token}`
    }
    // 添加请求时间戳防重放
    config.headers['X-Request-Time'] = Date.now().toString()
    return config
  },
  (error) => Promise.reject(error)
)

// 响应拦截器
request.interceptors.response.use(
  (response) => {
    const { code, data, message } = response.data
    if (code === 200 || code === 0) {
      return data
    }
    // 业务错误处理
    ElMessage.error(message || '请求失败')
    return Promise.reject(new Error(message))
  },
  async (error) => {
    const { response } = error
    if (response?.status === 401) {
      // Token过期，尝试刷新
      const userStore = useUserStore()
      try {
        await userStore.refreshAccessToken()
        return request(error.config)
      } catch {
        userStore.logout()
        window.location.href = '/login'
      }
    }
    ElMessage.error(response?.data?.message || '网络错误')
    return Promise.reject(error)
  }
)

export default request
```

### 3.7 WebSocket 集成

```typescript
// composables/useWebSocket.ts
export function useWebSocket() {
  const ws = ref<WebSocket | null>(null)
  const isConnected = ref(false)
  const reconnectCount = ref(0)
  const MAX_RECONNECT = 5
  
  const connect = (token: string) => {
    const wsUrl = `${import.meta.env.VITE_WS_URL}?token=${token}`
    ws.value = new WebSocket(wsUrl)
    
    ws.value.onopen = () => {
      isConnected.value = true
      reconnectCount.value = 0
      // 心跳
      startHeartbeat()
    }
    
    ws.value.onmessage = (event) => {
      const data = JSON.parse(event.data)
      handleMessage(data)
    }
    
    ws.value.onclose = () => {
      isConnected.value = false
      // 指数退避重连
      if (reconnectCount.value < MAX_RECONNECT) {
        const delay = Math.min(1000 * Math.pow(2, reconnectCount.value), 30000)
        setTimeout(() => {
          reconnectCount.value++
          connect(token)
        }, delay)
      }
    }
  }
  
  const handleMessage = (data: any) => {
    const notificationStore = useNotificationStore()
    switch (data.type) {
      case 'new_notification':
        notificationStore.addNotification(data.payload)
        break
      case 'unread_count_update':
        notificationStore.updateUnreadCount(data.payload)
        break
      case 'new_message':
        notificationStore.addMessage(data.payload)
        break
    }
  }
  
  return { ws, isConnected, connect }
}
```

```java
// service/UserService.java
public interface UserService extends IService<User> {
    
    UserVO register(RegisterDTO registerDTO);
    
    LoginVO login(LoginDTO loginDTO);
    
    UserVO getCurrentUser(Long userId);
    
    void updateProfile(Long userId, UpdateProfileDTO dto);
    
    void follow(Long userId, Long targetUserId);
    
    void unfollow(Long userId, Long targetUserId);
    
    Page<UserVO> getFollowingList(Long userId, PageParam pageParam);
    
    Page<UserVO> getFollowerList(Long userId, PageParam pageParam);
}

// service/impl/UserServiceImpl.java
@Service
@RequiredArgsConstructor
@Slf4j
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements UserService {
    
    private final UserMapper userMapper;
    private final RedisTemplate<String, Object> redisTemplate;
    private final JwtTokenUtil jwtTokenUtil;
    private final BCryptPasswordEncoder passwordEncoder;
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public UserVO register(RegisterDTO dto) {
        // 检查邮箱是否已注册
        if (lambdaQuery().eq(User::getEmail, dto.getEmail()).exists()) {
            throw new BusinessException(ErrorCode.EMAIL_ALREADY_EXISTS);
        }
        
        // 创建用户
        User user = User.builder()
                .email(dto.getEmail())
                .password(passwordEncoder.encode(dto.getPassword()))
                .nickname(dto.getNickname())
                .level(1)
                .points(0)
                .status(0)
                .build();
        
        save(user);
        
        // 发送欢迎通知（异步）
        eventPublisher.publishEvent(new UserRegisterEvent(this, user.getId()));
        
        return convertToVO(user);
    }
    
    @Override
    public LoginVO login(LoginDTO dto) {
        User user = userMapper.selectByEmail(dto.getEmail());
        if (user == null || !passwordEncoder.matches(dto.getPassword(), user.getPassword())) {
            throw new BusinessException(ErrorCode.INVALID_CREDENTIALS);
        }
        
        if (user.getStatus() != 0) {
            throw new BusinessException(ErrorCode.ACCOUNT_DISABLED);
        }
        
        // 生成Token
        String accessToken = jwtTokenUtil.generateAccessToken(user.getId());
        String refreshToken = jwtTokenUtil.generateRefreshToken(user.getId());
        
        // 缓存用户信息
        redisTemplate.opsForValue().set(
            RedisKey.USER_INFO + user.getId(), 
            user, 
            30, 
            TimeUnit.MINUTES
        );
        
        return LoginVO.builder()
                .userId(user.getId())
                .accessToken(accessToken)
                .refreshToken(refreshToken)
                .expiresIn(1800)
                .build();
    }
    
    @Override
    @Cacheable(value = "user", key = "#userId")
    public UserVO getCurrentUser(Long userId) {
        User user = getById(userId);
        if (user == null) {
            throw new BusinessException(ErrorCode.USER_NOT_FOUND);
        }
        return convertToVO(user);
    }
    
    @Override
    @Transactional(rollbackFor = Exception.class)
    public void follow(Long userId, Long targetUserId) {
        if (userId.equals(targetUserId)) {
            throw new BusinessException(ErrorCode.CANNOT_FOLLOW_SELF);
        }
        
        String followKey = RedisKey.USER_FOLLOWING + userId;
        String followerKey = RedisKey.USER_FOLLOWERS + targetUserId;
        
        // Redis原子操作
        Boolean isFollowing = redisTemplate.opsForSet().isMember(followKey, targetUserId);
        if (Boolean.TRUE.equals(isFollowing)) {
            throw new BusinessException(ErrorCode.ALREADY_FOLLOWING);
        }
        
        redisTemplate.opsForSet().add(followKey, targetUserId);
        redisTemplate.opsForSet().add(followerKey, userId);
        
        // 异步写入数据库
        eventPublisher.publishEvent(new FollowEvent(this, userId, targetUserId));
    }
    
    private UserVO convertToVO(User user) {
        return UserVO.builder()
                .id(user.getId())
                .email(user.getEmail())
                .nickname(user.getNickname())
                .avatar(user.getAvatar())
                .bio(user.getBio())
                .level(user.getLevel())
                .points(user.getPoints())
                .build();
    }
}
```

```java
// Gateway过滤器
@Component
@RequiredArgsConstructor
public class AuthFilter implements GlobalFilter, Ordered {
    
    private final JwtTokenUtil jwtTokenUtil;
    private final RedisTemplate<String, Object> redisTemplate;
    
    // 白名单路径
    private static final List<String> WHITE_LIST = Arrays.asList(
        "/users/login",
        "/users/register",
        "/articles",
        "/search"
    );
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String path = exchange.getRequest().getURI().getPath();
        
        // 白名单放行
        if (WHITE_LIST.stream().anyMatch(path::startsWith)) {
            return chain.filter(exchange);
        }
        
        // 获取Token
        String token = exchange.getRequest().getHeaders().getFirst("Authorization");
        if (token == null || !token.startsWith("Bearer ")) {
            return unauthorized(exchange);
        }
        
        token = token.substring(7);
        
        // 验证Token
        if (!jwtTokenUtil.validateToken(token)) {
            return unauthorized(exchange);
        }
        
        // 解析用户ID并传递给下游服务
        Long userId = jwtTokenUtil.getUserIdFromToken(token);
        ServerHttpRequest request = exchange.getRequest().mutate()
                .header("X-User-Id", String.valueOf(userId))
                .build();
        
        return chain.filter(exchange.mutate().request(request).build());
    }
    
    private Mono<Void> unauthorized(ServerWebExchange exchange) {
        ServerHttpResponse response = exchange.getResponse();
        response.setStatusCode(HttpStatus.UNAUTHORIZED);
        response.getHeaders().setContentType(MediaType.APPLICATION_JSON);
        
        Result<Void> result = Result.error(ErrorCode.UNAUTHORIZED);
        byte[] bytes = JSON.toJSONString(result).getBytes(StandardCharsets.UTF_8);
        
        return response.writeWith(Mono.just(response.bufferFactory().wrap(bytes)));
    }
    
    @Override
    public int getOrder() {
        return -100;
    }
}
```

```java
        "if isMember == 0 then " +
        "   return 0 " +
        "end " +
        "redis.call('srem', key, userId) " +
        "redis.call('decr', countKey) " +
        "return 1";

        String countKey = String.format("like:count:%d:%d", targetType, targetId);

            new DefaultRedisScript<>(LUA_LIKE_SCRIPT, Long.class),
            Arrays.asList(key, countKey),
            String.valueOf(userId)
        );
        
        if (result != null && result == 1) {
            // 发送异步消息持久化到数据库
            LikeMessage message = new LikeMessage(userId, targetId, targetType, true);
            rabbitTemplate.convertAndSend("like.exchange", "like.routing.key", message);
            return true;
        }
        return false;
    }

        String countKey = String.format("like:count:%d:%d", targetType, targetId);

            new DefaultRedisScript<>(LUA_UNLIKE_SCRIPT, Long.class),
            Arrays.asList(key, countKey),
            String.valueOf(userId)
        );
        
        if (result != null && result == 1) {
```

### 4.9 Feed流实现

```java
@Service
@RequiredArgsConstructor
public class FeedService {
    
    private final StringRedisTemplate redisTemplate;
    private final ArticleService articleService;
    private final UserService userService;
    
    // 大V阈值：粉丝数超过1万使用拉模式
    private static final int BIG_V_THRESHOLD = 10000;
    // Feed流最大长度
    private static final int FEED_MAX_SIZE = 500;
    
    /**
     * 发布文章时推送到粉丝Feed流（写扩散）
     */
    public void pushToFollowers(Long authorId, Long articleId, double score) {
        // 获取粉丝数
        Long followerCount = userService.getFollowerCount(authorId);
        
        // 大V不发推送，由拉模式获取
        if (followerCount > BIG_V_THRESHOLD) {
            return;
        }
        
        // 获取粉丝列表
        Set<Long> followers = userService.getFollowers(authorId);
        
        // 批量推送到粉丝Feed流
        for (Long followerId : followers) {
            String feedKey = String.format("feed:%d", followerId);
            redisTemplate.opsForZSet().add(feedKey, String.valueOf(articleId), score);
            
            // 控制Feed流长度
            Long size = redisTemplate.opsForZSet().size(feedKey);
            if (size != null && size > FEED_MAX_SIZE) {
                redisTemplate.opsForZSet().removeRange(feedKey, 0, size - FEED_MAX_SIZE - 1);
            }
        }
    }
    
    /**
     * 获取关注流（推拉结合）
     */
    public List<ArticleVO> getFollowingFeed(Long userId, Long cursor, int size) {
        List<Long> articleIds = new ArrayList<>();
        
        // 1. 从Feed流获取（推模式）
        String feedKey = String.format("feed:%d", userId);
        Set<ZSetOperations.TypedTuple<String>> feedItems;
        
        if (cursor == null || cursor == 0) {
            feedItems = redisTemplate.opsForZSet().reverseRangeWithScores(feedKey, 0, size - 1);
        } else {
            feedItems = redisTemplate.opsForZSet().reverseRangeByScoreWithScores(
                feedKey, 0, cursor - 1, 0, size
            );
        }
        
        if (feedItems != null) {
            for (ZSetOperations.TypedTuple<String> item : feedItems) {
                articleIds.add(Long.valueOf(item.getValue()));
            }
        }
        
        // 2. 补充大V的最新文章（拉模式）
        Set<Long> bigVIds = userService.getFollowingBigV(userId, BIG_V_THRESHOLD);
        if (!bigVIds.isEmpty()) {
            List<Long> bigVArticles = articleService.getLatestArticles(bigVIds, size / 2);
            articleIds.addAll(bigVArticles);
        }
        
        // 3. 去重并排序
        articleIds = articleIds.stream()
                .distinct()
                .limit(size)
                .collect(Collectors.toList());
        
        // 4. 批量获取文章详情
        return articleService.getArticlesByIds(articleIds);
    }
    
    /**
     * 获取推荐流（个性化推荐 + 热门内容）
     */
    public List<ArticleVO> getRecommendedFeed(Long userId, Long cursor, int size) {
        // 1. 获取个性化推荐（基于用户兴趣）
        List<Long> recommendedIds = getPersonalizedRecommendations(userId, size / 2);
        
        // 2. 获取热门内容
        List<Long> hotIds = getHotArticles(size / 4);
        
        // 3. 获取最新内容
        List<Long> latestIds = articleService.getLatestArticleIds(size / 4);
        
        // 4. 合并去重
        Set<Long> merged = new LinkedHashSet<>();
        merged.addAll(recommendedIds);
        merged.addAll(hotIds);
        merged.addAll(latestIds);
        
        List<Long> articleIds = merged.stream()
                .limit(size)
                .collect(Collectors.toList());
        
        return articleService.getArticlesByIds(articleIds);
    }
    
    private List<Long> getPersonalizedRecommendations(Long userId, int size) {
        // 基于用户标签和阅读历史的协同过滤推荐
        // 实际实现可调用推荐服务或使用Redis存储的推荐结果
        String recKey = String.format("rec:%d", userId);
        Set<String> recs = redisTemplate.opsForZSet().reverseRange(recKey, 0, size - 1);
        
        if (recs == null || recs.isEmpty()) {
            // 冷启动：返回热门内容
            return getHotArticles(size);
        }
        
        return recs.stream()
                .map(Long::valueOf)
                .collect(Collectors.toList());
    }
    
    private List<Long> getHotArticles(int size) {
        String hotKey = "hot:articles:daily";
        Set<String> hots = redisTemplate.opsForZSet().reverseRange(hotKey, 0, size - 1);
        
        if (hots == null) {
            return Collections.emptyList();
        }
        
        return hots.stream()
                .map(Long::valueOf)
                .collect(Collectors.toList());
    }
}
```

```java
                .withHighlightBuilder(highlightBuilder)

                .withSort(SortBuilders.scoreSort().order(SortOrder.DESC))
                .withSort(SortBuilders.fieldSort("createTime").order(SortOrder.DESC))

                    Map<String, List<String>> highlightFields = hit.getHighlightFields();

                    }
                    if (highlightFields.containsKey("summary")) {
                        vo.setSummary(highlightFields.get("summary").get(0));
                    }
                    
                    return vo;
                })
                .collect(Collectors.toList());
        
        return PageResult.of(list, searchHits.getTotalHits());
    }
```

### 4.11 热榜计算

```java
@Component
@RequiredArgsConstructor
public class HotRankCalculator {
    
    private final StringRedisTemplate redisTemplate;
    private final ArticleService articleService;
    
    /**
     * 热榜计算公式：Score = (P*2 + C*3 + S*5) / (T+2)^1.5
     * P: 点赞数，C: 评论数，S: 收藏数，T: 距今天数
     */
    @XxlJob("calculateHotRank")
    public void calculateHotRank() {
        // 获取最近7天的文章
        LocalDateTime sevenDaysAgo = LocalDateTime.now().minusDays(7);
        List<Article> articles = articleService.getArticlesAfter(sevenDaysAgo);
        
        // 计算每篇文章的热度分
        for (Article article : articles) {
            double score = calculateHotScore(article);
            
            // 存入日榜
            String dailyKey = "hot:articles:daily";
            redisTemplate.opsForZSet().add(dailyKey, 
                    String.valueOf(article.getId()), score);
            
            // 存入周榜
            String weeklyKey = "hot:articles:weekly";
            redisTemplate.opsForZSet().add(weeklyKey, 
                    String.valueOf(article.getId()), score);
            
            // 设置过期时间
            redisTemplate.expire(dailyKey, 2, TimeUnit.DAYS);
            redisTemplate.expire(weeklyKey, 8, TimeUnit.DAYS);
        }
        
        // 控制榜单长度
        redisTemplate.opsForZSet().removeRange(dailyKey, 0, -101);
        redisTemplate.opsForZSet().removeRange(weeklyKey, 0, -101);
    }
    
    private double calculateHotScore(Article article) {
        long likes = article.getLikeCount();
        long comments = article.getCommentCount();
        long favorites = article.getFavoriteCount();
        
        // 计算距今天数
        long days = ChronoUnit.DAYS.between(article.getCreateTime(), LocalDateTime.now());
        
        // 热度公式
        double numerator = likes * 2 + comments * 3 + favorites * 5;
        double denominator = Math.pow(days + 2, 1.5);
        
        return numerator / denominator;
    }
}
```

## 7. 安全策略

### 7.1 认证安全

- **密码加密**：BCrypt，强度10
- **Token策略**：
  - Access Token：有效期30分钟
  - Refresh Token：有效期7天，只能使用一次
- **登录安全**：
  - 连续5次失败锁定15分钟
  - 异地登录提醒
  - 单设备登录（可选）

### 7.2 接口安全

- **防重放攻击**：请求头携带时间戳，有效期5分钟
- **限流策略**：
  - 登录接口：5次/分钟
  - 普通接口：100次/分钟
  - 搜索接口：30次/分钟
- **敏感操作二次验证**：修改密码、绑定手机需要验证码

### 7.3 数据安全

- **传输加密**：全站HTTPS，TLS 1.3
- **敏感数据脱敏**：
  - 手机号：138****8888
  - 邮箱：a***@example.com
- **SQL注入防护**：预编译语句，参数化查询
- **XSS防护**：输入过滤，输出转义
