# 需求文档

## 1. 项目概述

构建一个专注于技术分享与交流的社区平台，核心功能类似稀土掘金，支持技术文章发布、社交互动、个性化推荐、全文搜索、实时消息通知等。系统采用前后端分离架构，适配 PC Web 和移动端 H5。

## 2. 用户角色

| 角色 | 说明 |
|------|------|
| 普通用户 | 浏览、发布、互动、关注 |
| 认证作者（可扩展） | 特殊标识，文章优先审核 |
| 管理员 | 内容审核、用户管理、运营配置 |
| 系统 | 自动化任务（审核、推荐、热榜计算） |

## 3. 功能需求详述

### 3.1 用户模块

#### 3.1.1 注册/登录
- **邮箱注册**：输入邮箱、密码、昵称，发送验证邮件激活账号
- **手机号注册**：输入手机号、验证码、密码、昵称
- **第三方登录**：支持 GitHub、微信 OAuth2 登录
- **密码找回**：通过邮箱或手机验证码重置密码
- **登录安全**：
  - 连续5次密码错误锁定账号15分钟
  - 异地登录检测和提醒
  - 支持单设备登录（可选）

#### 3.1.2 个人资料
- **基础信息**：头像、昵称、简介、个人主页背景图
- **技能标签**：最多选择5个技术标签
- **社交链接**：GitHub、个人博客、微博等
- **隐私设置**：
  - 是否公开收藏夹
  - 是否公开关注/粉丝列表
  - 是否允许陌生人私信

#### 3.1.3 成长体系
- **掘力值计算规则**：
  - 发布文章：+10分
  - 文章被点赞：+1分
  - 文章被收藏：+2分
  - 文章被评论：+1分
  - 评论被点赞：+0.5分
  - 每日签到：+1分
- **等级规则**：
  | 等级 | 掘力值范围 | 称号 |
  |------|------------|------|
  | Lv1 | 0-99 | 萌新 |
  | Lv2 | 100-499 | 初学者 |
  | Lv3 | 500-1999 | 进阶者 |
  | Lv4 | 2000-4999 | 熟练者 |
  | Lv5 | 5000-9999 | 专家 |
  | Lv6 | 10000+ | 大神 |
- **徽章系统**：
  - 连续签到7天/30天/100天
  - 首篇文章发布
  - 文章阅读量破千/万
  - 获得100/1000个赞

#### 3.1.4 关注系统
- **关注/取消关注**：点击关注按钮，实时更新关注状态
- **关注列表**：查看我关注的用户，按关注时间倒序
- **粉丝列表**：查看关注我的用户，支持移除粉丝
- **互相关注标识**：双方互相关注显示特殊标识

#### 3.1.5 账号安全
- **密码修改**：需要输入原密码验证
- **绑定/解绑**：支持绑定/解绑手机号、邮箱、第三方账号
- **账号注销**：
  - 需要验证身份（密码或手机验证码）
  - 账号进入30天冷静期，期间可撤销注销
  - 注销后数据保留90天后物理删除

### 3.2 内容模块

#### 3.2.1 文章发布
- **编辑器功能**：
  - Markdown 编辑器（基于 Monaco Editor）
  - 实时预览
  - 支持代码块、表格、图片、链接
  - 快捷键支持（Ctrl+S保存草稿）
- **文章设置**：
  - 标题（最多100字）
  - 摘要（自动提取或手动编辑，最多200字）
  - 封面图（支持本地上传或选择默认封面）
  - 标签（最多5个，支持搜索和新建）
  - 分类（前端、后端、移动端、AI等）
- **发布选项**：
  - 立即发布
  - 定时发布（选择未来时间）
  - 保存草稿
- **草稿箱**：
  - 自动保存（每30秒）
  - 最多保存50篇草稿
  - 草稿保留90天

#### 3.2.2 文章编辑
- **编辑权限**：仅作者本人可编辑
- **历史版本**：
  - 每次发布自动保存版本
  - 最多保留20个历史版本
  - 支持对比版本差异
  - 支持回滚到历史版本
- **编辑后状态**：
  - 已发布文章编辑后重新进入审核
  - 审核期间展示旧版本内容

#### 3.2.3 文章详情
- **内容展示**：
  - Markdown 渲染，代码高亮
  - 支持目录导航（基于标题层级）
  - 图片懒加载
  - 代码块复制功能
- **文章信息**：
  - 作者信息卡片
  - 发布时间、阅读量、点赞数、收藏数、评论数
  - 标签展示
  - 版权声明（默认保留权利）
- **相关推荐**：
  - 基于标签的相关文章推荐
  - 作者的其他文章
- **阅读体验**：
  - 字体大小调整
  - 日间/夜间模式
  - 全屏阅读模式

#### 3.2.4 标签体系
- **标签分类**：
  - 前端：Vue、React、JavaScript、CSS等
  - 后端：Java、Python、Go、Spring等
  - 移动端：iOS、Android、Flutter等
  - 其他：AI、DevOps、数据库等
- **标签功能**：
  - 标签详情页展示该标签下的热门文章
  - 关注标签，在推荐流中优先展示
  - 标签热度排行榜

#### 3.2.5 内容审核
- **审核流程**：
  ```
  草稿 -> 提交审核 -> 机器审核 -> [通过] -> 已发布
                            |
                            -> [可疑] -> 人工审核 -> [通过] -> 已发布
                                                    |
                                                    -> [拒绝] -> 驳回
  ```
- **机器审核**：
  - 敏感词过滤（政治、色情、暴力等）
  - 图片鉴黄/鉴暴
  - 垃圾内容识别
- **人工审核**：
  - 机器审核可疑内容进入人工审核队列
  - 用户举报内容优先审核
  - 审核时效：工作日24小时内
- **审核结果**：
  - 通过：文章正常展示
  - 驳回：发送通知给作者，说明原因，可修改后重新提交
  - 删除：严重违规直接删除，扣除掘力值，视情况封禁账号

#### 3.2.6 文章管理
- **文章状态**：
  - 草稿
  - 审核中
  - 已发布
  - 已驳回（显示驳回原因）
  - 已删除
- **删除策略**：
  - 作者删除：逻辑删除，保留数据
  - 管理员删除：逻辑删除，可申诉恢复
  - 自动清理：删除90天后物理删除

#### 3.2.7 专栏体系

- **专栏定义**：
  - 专栏是认证作者围绕特定主题创作的系列文章合集
  - 每个专栏有独立的封面、名称、简介
  - 专栏可被其他用户订阅，形成知识体系沉淀
- **创建专栏**：
  - 权限：认证作者可创建，普通用户不可创建
  - 数量限制：每位认证作者最多创建5个专栏
  - 信息填写：专栏名称（最多50字）、简介（最多500字）、封面图
- **专栏管理**：
  - 文章收录：作者可将自己的已发布文章加入专栏，并设置排序
  - 排序调整：支持拖拽排序，决定文章在专栏内的展示顺序
  - 专栏编辑：修改名称、简介、封面图
  - 专栏删除：专栏下无文章时可删除，有文章需先移除
- **专栏浏览**：
  - 专栏主页：展示专栏信息、文章列表（按排序展示）、订阅人数、阅读量
  - 文章内标识：文章详情页展示所属专栏入口，引导读者浏览专栏
  - 作者主页：展示该作者创建的专栏列表
- **专栏订阅**：
  - 订阅按钮：点击订阅专栏，再次点击取消订阅
  - 订阅列表：查看我订阅的专栏，按更新时间排序
  - 更新通知：订阅的专栏有新文章加入时，发送通知给订阅者
- **专栏发现**：
  - 热门专栏：按订阅数排序展示
  - 推荐专栏：基于用户兴趣标签推荐相关专栏
  - 分类筛选：按技术分类筛选专栏

### 3.3 社交模块

#### 3.3.1 点赞
- **点赞对象**：文章、评论、沸点
- **点赞功能**：
  - 点击点赞按钮，实时更新点赞数和状态
  - 再次点击取消点赞
  - 点赞用户列表（展示最近20个点赞用户头像）
- **点赞限制**：
  - 同一用户1分钟内最多点赞10次
  - 防止刷赞（异常行为检测）

#### 3.3.2 收藏
- **收藏夹管理**：
  - 默认收藏夹（不可删除）
  - 自定义收藏夹（最多创建20个）
  - 收藏夹名称、描述、公开/私密设置
- **收藏操作**：
  - 收藏时选择收藏夹（默认收藏到默认收藏夹）
  - 支持移动到其他收藏夹
  - 取消收藏
- **收藏展示**：
  - 个人主页展示公开收藏夹
  - 收藏夹内文章列表

#### 3.3.3 评论
- **评论功能**：
  - 支持富文本（表情、代码块、链接）
  - @用户（输入@弹出用户搜索）
  - 嵌套回复（最多2层）
- **评论管理**：
  - 作者可置顶精彩评论
  - 作者可删除评论
  - 用户可删除自己的评论
- **评论排序**：
  - 默认：按时间倒序
  - 可选：按热度（点赞数）排序
- **评论审核**：
  - 敏感词过滤
  - 用户举报后人工审核

#### 3.3.4 沸点
- **发布沸点**：
  - 文字（最多500字）
  - 图片（最多9张）
  - 链接（自动解析预览）
  - 话题标签 #话题#
- **沸点流**：
  - 推荐沸点（个性化推荐）
  - 关注沸点（关注用户的沸点）
  - 话题沸点（按话题聚合）
- **沸点互动**：
  - 点赞、评论
  - 转发（引用转发）
  - 分享

#### 3.3.5 分享
- **分享渠道**：
  - 复制链接
  - 生成分享卡片（带二维码）
  - 分享到微信、微博、QQ（调用对应SDK）
- **分享统计**：
  - 记录分享次数
  - 分享带来的阅读量统计

### 3.4 推荐模块

#### 3.4.1 首页推荐流
- **推荐内容组成**：
  - 关注流（我关注的用户发布的文章）
  - 个性化推荐（基于兴趣模型）
  - 热门内容（全站热门文章）
  - 编辑推荐（运营精选）
- **推荐算法（初期简化版）**：
  - 基于标签的协同过滤
  - 基于阅读历史的相似文章
  - 热门内容加权
  - 新文章冷启动保护（给予一定曝光）
- **冷启动策略**：
  - 新用户：展示全站热门内容
  - 引导选择感兴趣的标签
  - 根据选择的标签推荐相关内容
- **推荐反馈**：
  - 用户可标记"不感兴趣"
  - 减少相似内容推荐
  - 记录反馈优化推荐模型

#### 3.4.2 关注流
- **内容来源**：
  - 我关注的用户发布的文章
  - 我关注的用户发布的沸点（可选显示）
- **排序规则**：
  - 默认：按发布时间倒序
  - 智能排序：根据互动概率排序（可选）
- **大V处理**：
  - 粉丝数超过1万的用户（大V）
  - 大V内容不推送到粉丝Feed流
  - 粉丝主动访问大V主页或从推荐流看到

#### 3.4.3 热门榜单
- **榜单类型**：
  - 日榜（近24小时）
  - 周榜（近7天）
  - 月榜（近30天）
- **热度计算公式**：
  ```
  Score = (点赞数×2 + 评论数×3 + 收藏数×5 + 阅读数×0.1) / (发布天数+2)^1.5
  ```
- **榜单更新**：
  - 每小时计算一次
  - 缓存到Redis
  - 展示Top 100

### 3.5 搜索模块

#### 3.5.1 全局搜索
- **搜索范围**：
  - 文章（标题、摘要、正文）
  - 用户（昵称、简介）
  - 标签
  - 沸点
- **搜索建议**：
  - 输入时实时补全（基于热门搜索和历史搜索）
  - 搜索历史记录（本地存储，最近10条）
  - 热门搜索词（展示Top 10）
- **搜索结果**：
  - 高亮匹配关键词
  - 按相关度排序
  - 支持筛选（时间范围、类型）

#### 3.5.2 高级搜索
- **筛选条件**：
  - 时间范围：最近一天/一周/一月/一年/全部
  - 内容类型：文章/用户/标签/沸点
  - 排序方式：相关度/时间/热度
- **搜索语法**：
  - `tag:Java` 搜索指定标签
  - `author:张三` 搜索指定作者
  - `title:Spring` 只在标题中搜索

### 3.6 消息模块

#### 3.6.1 消息类型
| 类型 | 触发场景 | 通知方式 |
|------|----------|----------|
| 系统通知 | 账号安全提醒、等级提升、系统公告 | 站内信+推送 |
| 互动通知 | 文章被点赞/收藏/评论、被关注 | 站内信+推送 |
| @通知 | 评论中被@ | 站内信+推送 |
| 审核通知 | 文章审核通过/驳回 | 站内信+推送 |
| 私信 | 其他用户发来的私信 | 站内信+推送（需开启） |

#### 3.6.2 通知管理
- **通知列表**：
  - 按时间倒序展示
  - 支持按类型筛选
  - 未读消息标记红点
- **通知操作**：
  - 标记单条已读
  - 标记全部已读
  - 删除通知
- **免打扰设置**：
  - 可关闭某类通知的推送
  - 可设置免打扰时段

#### 3.6.3 私信功能
- **会话列表**：
  - 展示最近联系的用户
  - 显示最后一条消息预览
  - 未读消息数标记
- **私信对话**：
  - 实时消息（WebSocket）
  - 支持文字、表情、图片
  - 已读/未读状态
  - 消息撤回（2分钟内）
- **隐私设置**：
  - 允许所有人私信（默认）
  - 仅允许互相关注的人私信
  - 不允许任何人私信
- **黑名单**：
  - 拉黑后收不到对方消息
  - 可解除拉黑

#### 3.6.4 实时推送
- **推送方式**：
  - WebSocket 实时推送
  - 浏览器桌面通知（需用户授权）
  - 移动端推送（后续支持）
- **推送内容**：
  - 新消息通知
  - 未读消息数更新
  - 系统公告

### 3.7 运营模块

#### 3.7.1 签到系统
- **签到规则**：
  - 每日签到获得1掘力值
  - 连续签到7天额外奖励5掘力值
  - 连续签到30天额外奖励20掘力值
  - 断签后重新计算
- **签到记录**：
  - 日历形式展示签到记录
  - 补签卡（运营活动发放）

#### 3.7.2 任务中心
- **新手任务**：
  - 完善个人资料（+5掘力值）
  - 发布第一篇文章（+10掘力值）
  - 获得第一个赞（+5掘力值）
  - 关注3个感兴趣的用户（+5掘力值）
- **日常任务**：
  - 每日签到（+1掘力值）
  - 阅读5篇文章（+2掘力值）
  - 点赞3篇文章（+2掘力值）
  - 评论1篇文章（+3掘力值）
- **任务奖励**：
  - 掘力值
  - 徽章
  - 积分（用于积分商城，后续扩展）

#### 3.7.3 广告管理
- **广告位**：
  - 首页Banner（轮播，最多5个）
  - 文章详情页侧边栏
  - 信息流广告（每隔10篇文章插入1条）
- **广告类型**：
  - 图片广告
  - 文字链广告
- **广告控制**：
  - 用户可关闭不感兴趣的广告
  - 广告点击率统计

#### 3.7.4 数据统计（管理后台）
- **用户数据**：
  - 日活/月活用户
  - 新增用户趋势
  - 用户留存率
- **内容数据**：
  - 文章发布量
  - 文章阅读量趋势
  - 热门标签统计
- **互动数据**：
  - 点赞/收藏/评论数
  - 用户互动排行

#### 3.7.5 系统配置

- **配置管理**：
  - 系统级参数通过管理后台配置，无需修改代码或重启服务
  - 配置以 Key-Value 形式存储，按分组归类
  - 配置变更实时生效（通过 Redis 发布订阅刷新缓存）
- **配置分组**：
  | 分组 | 示例配置项 | 说明 |
  |------|-----------|------|
  | `content` | 文章自动保存间隔、草稿保留天数、每日发文上限 | 内容相关配置 |
  | `social` | 评论最大字数、点赞频率限制、沸点最大字数 | 社交相关配置 |
  | `user` | 注册开关、默认头像、掘力值规则参数 | 用户相关配置 |
  | `audit` | 敏感词过滤开关、图片审核开关、人工审核阈值 | 审核相关配置 |
  | `system` | 全站维护模式、公告弹窗、SEO关键词 | 系统级配置 |
- **配置操作**：
  - 查询：按分组查配置列表，支持模糊搜索
  - 修改：修改配置值和描述说明
  - 新增：添加新的自定义配置项
  - 删除：删除不再使用的配置项（仅系统管理员）
  - 回滚：保留最近10次变更记录，支持回滚到历史值
- **配置安全**：
  - 敏感配置（如密钥、Token）加密存储，展示时脱敏
  - 配置变更需二次确认
  - 所有变更记录操作日志，可追溯

#### 3.7.6 操作日志

- **日志范围**：
  - 管理员操作：审核文章、封禁用户、修改配置、删除内容等
  - 用户敏感操作：注销账号、修改密码、绑定解绑等
  - 系统自动操作：定时任务执行、自动审核结果等
- **日志内容**：
  | 字段 | 说明 |
  |------|------|
  | 操作人 | 用户ID和用户名（系统操作为"system"） |
  | 操作类型 | 如：审核文章、封禁用户、修改配置 |
  | 操作描述 | 可读的操作详情 |
  | 请求信息 | 请求方法、URL、参数（脱敏处理） |
  | 响应结果 | 操作成功/失败状态码 |
  | 执行耗时 | 毫秒级 |
  | 客户端信息 | IP地址、User-Agent |
  | 操作时间 | 精确到毫秒 |
- **日志查询**：
  - 时间范围筛选（精确到小时）
  - 按操作人、操作类型筛选
  - 支持关键词搜索（请求参数、操作描述）
  - 分页查询，默认按时间倒序
- **数据保留**：
  - 操作日志保留180天，超过期限自动归档
  - 归档数据可导出为CSV/JSON文件
  - 敏感操作的日志不可删除，永久保留
- **日志导出**：
  - 支持按时间范围导出
  - 导出格式：CSV（Excel兼容）或 JSON
  - 导出需管理员权限，且导出操作本身也记录日志

### 3.8 资源模块

#### 3.8.1 文件上传
- **上传方式**：
  - 前端直传OSS（阿里云/MinIO）
  - 服务端获取临时上传凭证
  - 上传成功后回调服务端
- **文件类型**：
  - 图片：jpg、png、gif、webp，单张最大5MB
  - 附件：pdf、doc、docx、zip，单个最大50MB
- **图片处理**：
  - 自动生成多尺寸缩略图
  - 图片压缩
  - 添加水印（可选）
  - 违规图片自动拦截

#### 3.8.2 CDN加速
- **静态资源**：
  - JS/CSS文件
  - 图片资源
  - 字体文件
- **CDN配置**：
  - 全站静态资源CDN分发
  - 缓存策略（JS/CSS长期缓存，图片中期缓存）

#### 3.8.3 图片懒加载
- **懒加载策略**：
  - 文章详情页图片进入视口前加载
  - 使用占位图（低质量预览图或骨架屏）
  - 加载失败显示默认错误图
